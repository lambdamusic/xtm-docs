read_and_write_frame   xtlang

Defined in:  https://github.com/digego/extempore/tree/v0.8.9/examples/contrib/opencv_test.xtm

Implementation
(bind-func read_and_write_frame
  (lambda (vc vw:ocv_VideoWriter* img)
    (let ((frame (ocv_read vc img)))
      (ocv_imshow 'Video' frame)
      (ocv_write vw frame)
      (ocv_waitKey 1))))
