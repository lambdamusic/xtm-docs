FBO_create   xtlang

Defined in:  https://github.com/digego/extempore/tree/v0.8.9/libs/external/gl/gl-objects.xtm

Implementation
(bind-func FBO_create
  (lambda ()
    (let ((fbo:FBO* (halloc))
          (id:GLuint* (salloc)))
      (glGenFramebuffers 1 id)
      (gl_print_error "Error creating FBO")
      (tset! fbo 0 (pref id 0))
      fbo)))
