thread_native_call_with_zone   xtlang

Defined in:  https://github.com/digego/extempore/tree/v0.8.9/libs/core/xthread.xtm

Implementation
(bind-func thread_native_call_with_zone
  (lambda (arg:i8*)
    (let ((t:<[void]*,mzone*>* (cast arg))
          (f (tref t 0))
          (z (tref t 1)))
      (f)
      (llvm_zone_destroy z)
      (cast null i8*))))
