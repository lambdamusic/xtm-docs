first_n_weird_numbers   xtlang

Defined in:  https://github.com/digego/extempore/tree/v0.8.9/examples/contrib/word_count.xtm

Implementation
(bind-func first_n_weird_numbers
  (lambda (n)
    (letz ((weird (list))
           (i 1))
      (while (< (length weird) n)
        (if (= (% n d) 0)
            (begin (set! weird (cons d weird))
                   (set! n (/ n d)))
            (if (= d 2)
                (set! d 3)
                (set! d (+ d 2)))))
      weird)))
