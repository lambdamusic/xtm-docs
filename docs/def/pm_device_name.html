pm_device_name   xtlang

Defined in:  https://github.com/digego/extempore/tree/v0.8.9/examples/sharedsystem/midisetup.xtm

Implementation
(bind-func pm_device_name
  (lambda (device_id)
    (let ((dev_info (Pm_GetDeviceInfo device_id)))
      (if (null? dev_info)
          "" ;; "empty" name
          (tref dev_info 2)))))
