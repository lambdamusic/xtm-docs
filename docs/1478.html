fill_direction_vector   xtlang

Implementation
(bind-func fill_direction_vector
  (lambda (v1:float* v2:float* v3:float*)
    (let ((tmp:float* (salloc 4)))
      (vsub v1 v2 3 tmp)
      (vnorm tmp 3 v3)
      (pset! v3 3 0.0) ;; direction vectors get w = 0.0
      void)))
