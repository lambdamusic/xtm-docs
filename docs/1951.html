glfw_init_and_create_interaction_compatibility_window   xtlang

Defined in:  https://github.com/digego/extempore/tree/v0.8.9/libs/external/glfw3.xtm

Implementation
(bind-func glfw_init_and_create_interaction_compatibility_window
  (lambda (width height)
    (glfw_init)
    ;; (glfw_set_compatibility_profile)
    (let ((window (glfw_create_interaction_window width height)))
      (glfwShowWindow window)
      (glfwMakeContextCurrent window)
      (glfwSwapInterval 1)
      (glfwSwapBuffers window)
      window)))
